Welcome to the **uniR API** — your gateway to seamless university management as a service.

This guide will help you set up your account, get an API key, and make your first call in minutes.

---

## 1. Introduction

The uniR API lets you programmatically manage university systems, including:

- Students
- Enrollment
- Courses & schedules
- Grades
- Staff

## 2. Prerequisites

Before you begin, ensure you have:

- An active [uniR API account](#) (sign up on our dashboard)
- Your API key or token
- Familiarity with REST and JSON (recommended)

---
It’s designed for easy integration into your apps, platforms, or internal tools.

## 3. Base URL

All API requests are made to: https://api.unir.com/v1
